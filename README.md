# Gestore di Prenotazioni

E' un app in python fatta con la libreria tkinter, simula un gestore di prenotazioni.

## Prerequisiti

- Tkinter: Libreria
- Python

## Come utilizzare

1. **Avvio dello script:**

   - Esegui lo script con il comando:
     ```bash
     python main.py
     ```

2. **Scegli un opzione:**

   - **1**: Prenota
   - **2**: Impostazioni

3. **(Prenota) Seleziona un'opzione:**

   - **1**: Sala 1.
   - **2**: Sala 2.
   - **3**: Sala 3.
   - **4**: Sala 4.

4. **(Impostazioni) Seleziona un'opzione:**

5. **Uscita:**
   - Puoi uscire dal'app con i Button in alto a sinistra.
